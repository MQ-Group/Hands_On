# 工具与资源

这里收录了课题组常用的工具和资源链接。

## 开发工具

### 深度学习框架
- **PyTorch**: 主流的深度学习框架
- **JAX**: 高性能机器学习研究框架

### 硬件仿真工具
- **Gem5**: 计算机系统(CPU)仿真器
- **Chisel**: 硬件描述语言
- **Vivado**: Xilinx FPGA 开发工具

## 学术资源

### 会议与期刊
- [ISCA](https://iscaconf.org/): 计算机体系结构国际会议
- [MICRO](https://www.microarch.org/): 微体系结构国际会议
- [ASPLOS](https://asplos-conference.org/): 体系结构支持编程语言和操作系统会议
- [HPCA](https://hpca-conf.org/): 高性能计算机体系结构国际会议
